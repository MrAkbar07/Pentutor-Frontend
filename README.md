# Pentutor

Pentutor is a full-stack educational platform built to simplify and enhance the tutoring experience for both students and mentors. Designed for developers, cybersecurity learners, and tech enthusiasts, Pentutor provides seamless scheduling, interactive lessons, and mentor-matching features.

## 🚀 Features

- 🧑‍🏫 Mentor & Student Profiles
- 📅 Session Booking and Scheduling
- 💬 Live Chat & Messaging System
- 📚 Course & Resource Management
- 🎓 Progress Tracking and Feedback
- 🔒 Secure Authentication (JWT/OAuth)
- 📈 Admin Dashboard with Analytics

## 🛠️ Tech Stack

- **Frontend:** React.js / Tailwind CSS
- **Backend:** Node.js / Express.js / Django / Laravel
- **Database:** MongoDB / PostgreSQL / MySQL
- **Authentication:** JWT, OAuth2
- **Deployment:** Docker, Vercel/Netlify, Heroku/DigitalOcean
